This package contains classes for the Telescope Position Editor.

Usage: java TelescopePositionEditor [-internalframes] [-shownavigator] [imageFileOrUrl]

The jsky.catalog.skycat.config property defines the Skycat style catalog config file to use.

If the -internalframes option is given, the application uses Windows style internal frames.

If -shownavigator is specified, the catalog navigator window is displayed on startup.

The imageFileOrUrl argument may be an image file or URL to load.
