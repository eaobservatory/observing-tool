Building the UKIRT OT

To build the UKIRT OT, run ant in the OT directory. The live copy is in /jac_sw/orac3/: 

/jac_sw/orac3 $ ant
...
BUILD SUCCESSFUL
Total time: 5 seconds

Reminders

  • Builds in the live system are only done Monday–Thursday daytime excluding holidays.
  • Since you will be working in a shared directory you obviously need to set your umask to 2 first.
  • If the build fails you will need to fix it before you can run any software which depends on it, such as observing normally at UKIRT using the QT or UKIRT Translator — the OT and QT should both be tested.
